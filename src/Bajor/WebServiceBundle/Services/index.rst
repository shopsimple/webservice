EquationService
------------------------------

Service with quadratic api to resolve quadratic equation.

Params:

* float $a
* float $b
* float $c
* float $minx
* float $maxx
      
Returns:

* array
  
  * float delta
  * string deltaInfo
  * float re_1 OPTIONAL
  * float re_2 OPTIONAL
  * float im_1 OPTIONAL
  * float im_2 OPTIONAL
  * float r_1 OPTIONAL
  * float r_2 OPTIONAL
  * float r_0 OPTIONAL
  * array solutions: int x => float y
